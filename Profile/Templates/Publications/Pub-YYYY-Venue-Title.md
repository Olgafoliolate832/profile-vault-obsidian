## How to Use

1. Fill in all frontmatter fields — `title`, `pub_type`, `authors`, `venue`, `year`, `status` at minimum.
2. Set `pub_type` to one of: `journal`, `conference`, `book-chapter`, `book`, `report`, `preprint`, `thesis`, `software`, `article`, `blog`.

4. Set `peer_reviewed` to `true` or `false`.
5. Add DOI and URL when available.
6. Leave `zotero_key` blank until Zotero integration is configured.
