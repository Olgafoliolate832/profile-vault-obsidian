# Profile

> This is the root overview of the Profile Vault.

This vault is a modular knowledge base for managing your professional identity, career history, and applications. Every top-level folder has a matching overview file that serves as its entry point and dashboard.

## Rules

1. **This file is the single entry point** to the entire vault. Keep it updated when adding new top-level folders.
2. **Every top-level folder** must be linked from here.
3. **Use wikilinks** to link to each folder's overview file (e.g., `[[Applications]]`, `[[Skills]]`).
4. **Keep descriptions brief** — one line per folder. Detailed documentation lives inside each folder.

## Knowledge Areas

- [[Applications]] — Active and past applications (Jobs, Grants, Fellowships, PhDs, Conferences)
- [[Experience]] — Professional roles across Academic, Industry, and Freelance
- [[Education]] — Degrees, certifications, and continuing education
- [[Skills]] — Technical, research, consulting, languages, and software
- [[Publications]] — Journals, conferences, books, reports
- [[Projects]] — Notable projects and their outcomes
- [[Grants-Funding]] — Funded and submitted grants
- [[Awards-Honors]] — Awards, prizes, and recognitions
- [[Talks-Workshops]] — Conference talks, invited lectures, workshops
- [[Professional-Service]] — Reviewing, committees, mentoring
- [[References]] — Referees and their contact details
- [[Identity]] — Bios, personal info, research/teaching statements
- [[Archive]] — Outdated or retired content
- [[Templates]] — Folder structures and file templates for all of the above
