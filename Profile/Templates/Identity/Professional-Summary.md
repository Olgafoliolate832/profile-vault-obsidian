# Professional Summary

> *Parent: [[Identity]]*

Domain-specific elevator pitches. Maintain separate versions for each professional domain you operate in, so Claude can pull the right one for each application.

## How to Use

1. Write a 3-5 sentence summary for each domain you work in.
2. Focus on what you do, what you're known for, and what value you bring.
3. Update when your focus or achievements shift.
4. These feed directly into CVs and cover letters — keep them sharp.
