This is the entry point for a single PhD application. Fill in the frontmatter fields above when creating a new application. This file is queried by the Applications Dashboard.

## How to Use

1. Fill in all frontmatter fields — especially `university`, `department`, `supervisor`, `deadline`, and `status`.
2. Update `status` as the application progresses: `researching` → `drafting` → `submitted` → `interview` → `offered` / `accepted` / `rejected` / `withdrawn`.
3. Log key actions in the Timeline table below.
4. Contact potential supervisors early — note correspondence in Notes.
5. Coordinate reference letters — link referees from the References folder.
6. Link deliverables in the Outputs section once created.
