# Requirements

> *Parent: [[Context]]*

Extract academic, research, and administrative requirements from the posting. Understanding all requirements early prevents last-minute surprises.

## Rules

- Separate academic requirements (degree, GPA, field) from research requirements (experience, methods, topics).
- Note all required documents — transcripts, writing samples, etc. take time to prepare.
- Check language requirements early — some require official certificates (IELTS, TOEFL, etc.).
- Note any deadlines for separate components (e.g., reference letters submitted separately).
