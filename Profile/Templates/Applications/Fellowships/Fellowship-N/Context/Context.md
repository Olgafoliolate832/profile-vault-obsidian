# Context

> *Parent: [[Index]]*

This folder contains all information **about the fellowship program** — what they offer, how they select, and what they value. This is input material that you gather before strategizing.

## How to Use

1. Start with [[Description]] — paste or summarize the fellowship program details.
2. Extract eligibility and selection criteria in [[Requirements]].
3. Research the organization in [[Research]] — past fellows, host institution, and strategic fit.

## Context Documents

- [[Description]] — the fellowship program
- [[Requirements]] — eligibility, selection criteria, and required documents
- [[Research]] — organization, past fellows, and host institution
