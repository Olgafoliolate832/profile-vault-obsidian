# Fellowship Application — [Organization] — [Program]

> *Parent: [[Fellowships]]*

This is the entry point for a single fellowship application. Fill in the frontmatter fields above when creating a new application. This file is queried by the Applications Dashboard.

## How to Use

1. Fill in all frontmatter fields — especially `organization`, `program`, `location`, `deadline`, and `status`.
2. Update `status` as the application progresses: `researching` → `drafting` → `submitted` → `interview` → `offered` / `accepted` / `rejected` / `withdrawn`.
3. Log key actions in the Timeline table below.
4. Coordinate reference letters early — link referees from the References folder.
5. Link deliverables in the Outputs section once created.
