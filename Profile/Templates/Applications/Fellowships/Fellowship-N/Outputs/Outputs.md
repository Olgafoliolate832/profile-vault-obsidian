# Outputs

> *Parent: [[Index]]*

This folder contains the **deliverables** — the actual documents you submit. Generate these after completing Context and Strategy.

## How to Use

1. Complete Context and Strategy first — never draft blind.
2. Use the Mapping from Strategy to drive content for each deliverable.
3. Pull from your profile vault folders (Experience, Skills, Publications, Projects) via wikilinks.
4. Coordinate reference letters early — link referees from the References folder.
5. Name output files descriptively (e.g., `CV.md`, `Research-Statement.md`, `Cover-Letter.md`).
6. Store non-Markdown outputs (PDFs, Word docs) in an `Assets/` subfolder if needed.

## Typical Deliverables

- CV / Résumé
- Research Statement / Research Plan
- Cover Letter / Motivation Letter
- Reference Letters (coordinated via References folder)
- Host institution confirmation (if required)
