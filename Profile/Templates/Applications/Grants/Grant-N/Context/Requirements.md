# Requirements

> *Parent: [[Context]]*

Extract eligibility criteria, evaluation criteria, and required documents from the funding call. Understanding how proposals are scored is essential for a competitive application.

## Rules

- Separate **eligibility** (must-have to even apply) from **evaluation criteria** (how proposals are scored).
- Note the weighting if provided (e.g., "Scientific excellence: 40%, Impact: 30%").
- List all required documents — missing documents can disqualify you.
- Note funding priorities — what does this funder especially care about right now?
