# Outputs

> *Parent: [[Index]]*

This folder contains the **deliverables** — the actual documents you submit to the funder. Generate these after completing Context and Strategy.

## How to Use

1. Complete Context and Strategy first — never draft blind.
2. Use the Mapping from Strategy to drive content for each deliverable.
3. Pull track record from Grants-Funding, Publications, and Projects folders via wikilinks.
4. Name output files descriptively (e.g., `Grant-Profile.md`, `Budget-Summary.md`).
5. Store Excel budgets in `Resources/` and link from here.
6. Store non-Markdown outputs (PDFs, Word docs) in an `Assets/` subfolder if needed.

## Typical Deliverables

- Grant profile / PI profile
- Budget summary (linked to Excel spreadsheet)
- Project description / proposal narrative
- Work plan / Gantt chart
- Letters of support
- Ethics statement (if applicable)
