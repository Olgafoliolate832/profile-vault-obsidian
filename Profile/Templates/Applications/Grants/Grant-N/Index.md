# Grant Application — [Funder] — [Title]

> *Parent: [[Grants]]*

This is the entry point for a single grant application. Fill in the frontmatter fields above when creating a new application. This file is queried by the Applications Dashboard.

## How to Use

1. Fill in all frontmatter fields — especially `funder`, `program`, `amount`, `role`, `deadline`, and `status`.
2. Update `status` as the application progresses: `researching` → `drafting` → `submitted` → `accepted` / `rejected` / `withdrawn`.
3. Log key actions in the Timeline table below.
4. Link deliverables in the Outputs section once created.
5. Store detailed budget spreadsheets in Excel under `Resources/` and link from Outputs.
