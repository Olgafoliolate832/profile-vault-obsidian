This is the entry point for a single conference submission. Fill in the frontmatter fields above when creating a new submission. This file is queried by the Applications Dashboard.

## How to Use

1. Fill in all frontmatter fields — especially `conference`, `track`, `deadline`, and `status`.
2. Update `status` as the submission progresses: `researching` → `drafting` → `submitted` → `accepted` / `rejected` / `withdrawn`.
3. Log key actions in the Timeline table below.
4. Note co-authors and their roles in Notes.
5. Link deliverables in the Outputs section once created.
