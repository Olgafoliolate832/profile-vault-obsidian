# Outputs

> *Parent: [[Index]]*

This folder contains the **deliverables** — the actual documents you submit. Generate these after completing Context and Strategy.

## How to Use

1. Complete Context and Strategy first — never draft blind.
2. Use the Mapping from Strategy to drive content for each deliverable.
3. Pull from your profile vault folders (Publications, Projects, Research) via wikilinks.
4. Name output files descriptively (e.g., `Abstract.md`, `Bio.md`, `Full-Paper.md`).
5. Follow formatting and citation requirements from Requirements.md exactly.
6. Store non-Markdown outputs (PDFs, LaTeX files) in an `Assets/` subfolder if needed.

## Typical Deliverables

- Abstract
- Full paper (if applicable)
- Speaker bio
- Presentation slides (post-acceptance)
- Supplementary materials
