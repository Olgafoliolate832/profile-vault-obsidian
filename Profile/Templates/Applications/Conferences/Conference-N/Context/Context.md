# Context

> *Parent: [[Index]]*

This folder contains all information **about the conference** — the call for papers, submission guidelines, and the conference itself. This is input material that you gather before strategizing.

## How to Use

1. Start with [[Description]] — paste or summarize the call for papers.
2. Extract topics, guidelines, and review criteria in [[Requirements]].
3. Research the conference in [[Research]] — reputation, past proceedings, and audience.

## Context Documents

- [[Description]] — the call for papers or submissions
- [[Requirements]] — topics of interest, submission guidelines, and review criteria
- [[Research]] — conference reputation, audience, key speakers, and past proceedings
