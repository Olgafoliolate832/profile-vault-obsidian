This is the entry point for a single job application. Fill in the frontmatter fields above when creating a new application. This file is queried by the Applications Dashboard.

## How to Use

1. Fill in all frontmatter fields — `title`, `organization`, `role`, `deadline`, `status`, `priority`, and `tags` at minimum.
2. Update `status` as the application progresses: `researching` → `drafting` → `submitted` → `interview` → `offered` / `accepted` / `rejected` / `withdrawn`.
3. Log key actions in the Timeline table below.
4. Link deliverables in the Outputs section once created.
